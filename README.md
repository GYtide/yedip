# yedip

数字图像处理实验作业

原生 js + html + css + electron 实现简单的BMP图片处理

包括
0. 读取并显示 BMP 格式图片
1. 灰度化
2. 反色
3. 显示直方图
4. 直方图均衡化
5. 几种简单锐化
6. 平滑
7. 添加噪声
8. 二值化处理
